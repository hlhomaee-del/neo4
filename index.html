Exam thinking moves

Read the encounter reason first. Ask why the patient is here today.
Look for sequencing triggers. Treatment only, transplant, pregnancy, eradicated primary.
Separate active disease from history. Active conditions sequence before history codes.
Check whether the cancer is primary or secondary. Do not confuse metastasis with a primary site code.

Common traps

Picking the right code but placing it in the wrong sequence.
Using a primary cancer code when the case documents metastasis.
Using a personal history code before an active secondary malignancy.
Using laterality or sex-specific codes when documentation does not support them.
Memorizing code numbers without understanding the concept that drives the order.
